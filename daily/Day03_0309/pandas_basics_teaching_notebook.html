Pandas Basics#

pandas - Python Data Analysis Library#

https://pandas.pydata.org

This notebook introduces core Pandas concepts for beginners, including:

  • Creating Series and DataFrame objects

  • Reading and inspecting data

  • Selecting rows and columns

  • Filtering data

  • Adding new columns

  • Basic statistics

  • Grouping and aggregation

  • Sorting and handling missing values

by GPT 5.2

1. Import Pandas#

Pandas is the standard Python library for working with tabular data.

import pandas as pd

print(pd.__version__)

2. Create a Series#

A Series is like a labeled one-dimensional array.

temperatures = pd.Series([72, 75, 78, 74], index=['Mon', 'Tue', 'Wed', 'Thu'])
temperatures

3. Create a DataFrame#

A DataFrame is a table with rows and columns.

data = {
    'name': ['Alice', 'Bob', 'Charlie', 'Diana', 'Ethan'],
    'age': [23, 25, 22, 24, 26],
    'major': ['Physics', 'Math', 'Biology', 'Chemistry', 'Math'],
    'score': [88, 92, 85, 91, 79]
}

df = pd.DataFrame(data)
df

4. Inspect the DataFrame#

Useful methods for looking at your data quickly.

df.head()

df.info()

df.describe()

5. Select Columns#

You can select one column or multiple columns.

df['name']

df[['name', 'score']]

6. Select Rows with .iloc and .loc#

  • .iloc uses integer positions

  • .loc uses labels

df.iloc[0]

df.iloc[1:3, ['name', 'major', 'score']]

df.loc[2]

7. Filter Rows#

Boolean conditions let you keep only rows that match a rule.

df[df['score'] >= 90]

df[df['major'] == 'Math']

df[(df['score'] >= 85) & (df['age'] < 25)]

8. Add a New Column#

You can create columns from existing columns.

df['passed'] = df['score'] >= 85
df

df['score_plus_5'] = df['score'] + 5
df

9. Basic Statistics#

Pandas makes column-based calculations easy.

df['score'].mean(), df['score'].max(), df['score'].min()

df[['age', 'score']].mean()

10. Sort Values#

Sorting is useful when ranking or organizing data.

df.sort_values('score', ascending=False)

11. Group By#

Group rows by a category and calculate summary values.

df.groupby('passed')['score'].mean()

df.groupby('passed').agg({
    'score': ['mean', 'max'],'age': 'mean'
})

12. Missing Values#

Real datasets often contain missing data.

df2 = df.copy()
df2.loc[2, 'score'] = None
df2.loc[4, 'major'] = None
df2

df2.isna()

df2.dropna()

df2['score'] = df2['score'].fillna(df2['score'].mean())
df2['major'] = df2['major'].fillna('Unknown')
df2

13. Read and Write CSV Files#

These are among the most common Pandas tasks.

# Save a DataFrame to CSV
df.to_csv('students.csv', index=False)

# Read it back
students = pd.read_csv('students.csv')
students.head()

14. Practice Exercises#

Try these on your own:

  1. Select only the name and major columns.

  2. Filter students with score < 90.

  3. Add a new column called age_next_year.

  4. Find the average score for each major.

  5. Sort the DataFrame by age from smallest to largest.

15. Summary#

In this notebook, you practiced how to:

  • Build Series and DataFrame objects

  • Inspect data with head(), info(), and describe()

  • Select and filter data

  • Create new columns

  • Compute statistics

  • Group, sort, and clean data

  • Read from and write to CSV files

These are the core Pandas skills used in many data analysis workflows.
