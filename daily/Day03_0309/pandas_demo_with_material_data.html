Pandas Demonstration Notebook#

This notebook demonstrates several commonly used Pandas functions for data analysis.

In this notebook we will learn how to:

  • Load and inspect a dataset

  • Explore the structure of a DataFrame

  • Handle missing data

  • Identify duplicate records

  • Filter and select data

  • Perform simple data analysis

This code and data are adopted from Kaggle repository, Materials and their Mechanical Properties

Create additional information to the original data#

  • Let’s take a quick look of the relationship between ultimate strength and yield strength by making a scatter plot.

  • Use plt.scatter() function.

plt.scatter(x='Su', y='Sy', data=df)

  • Write a rating condition to rate the materials.

def get_rating(row):
    conditions = [
        (438.3 <= row['Su'] <= 535.7, 318.6 <= row['Sy'] <= 389.4, 204930 <= row['E'] <= 209070, 71100 <= row['G'] <= 86900, 0.285 <= row['mu'] <= 0.315, 7467 <= row['Ro'] <= 8253),
        (389.6 <= row['Su'] <= 584.4, 283.2 <= row['Sy'] <= 424.8, 202860 <= row['E'] <= 211140, 63200 <= row['G'] <= 94800, 0.27 <= row['mu'] <= 0.33, 7074 <= row['Ro'] <= 8646),
        (340.9 <= row['Su'] <= 633.1, 247.8 <= row['Sy'] <= 460.2, 200790 <= row['E'] <= 213210, 55300 <= row['G'] <= 102700, 0.255 <= row['mu'] <= 0.345, 6681 <= row['Ro'] <= 9039),
        (292.2 <= row['Su'] <= 681.8, 212.4 <= row['Sy'] <= 495.6, 198720 <= row['E'] <= 215280, 47400 <= row['G'] <= 110600, 0.24 <= row['mu'] <= 0.36, 6288 <= row['Ro'] <= 9432)
    ]
    ratings = [5, 4, 3, 2]

    for rating, condition in zip(ratings, conditions):
        if all(condition):
            return rating
    return 1

Use the function to rate materials and add the ratings to a new column of the data frame.

df['rating'] = df.apply(get_rating, axis=1)
df

  • The text label can be converted to numerical values.

df['Use'] = df['Use'].astype(int)

df['Su/Sy'] = df['Su'] / df['Sy']
