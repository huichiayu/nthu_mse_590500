Introduction to NumPy#

Audience: students who know basic Python (variables, lists, loops)

Goals:

  • Create NumPy arrays

  • Understand shape / ndim / dtype

  • Index and slice arrays

  • Use vectorized operations

  • Understand broadcasting

  • Do basic matrix operations

  • Compare speed vs Python lists

by GPT 5.2

0. Setup#

import numpy as np

1. Why NumPy?#

Python lists don’t do elementwise math by default. NumPy arrays do.

Python list example#

a = [1, 2, 3]
b = [4, 5, 6]
a + b  # This concatenates lists, not elementwise addition

NumPy array example#

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
c = a + b  # Elementwise addition
c

Quick thought: Why might vectorization matter in scientific computing?

2. Creating Arrays#

From Python lists#

arr = np.array([1, 2, 3, 4])
arr

Constructors#

np.zeros(5)

np.ones((2, 3))

3. Array Attributes#

Key ideas: shape (size in each dimension), number of dimensions, total size, data type.

A = np.array([[1, 2.0, 3],
              [4, 5, 6]])
A

A.shape

A.ndim

A.size

A.dtype

4. Indexing and Slicing#

A = np.array([[10, 20, 30],
              [40, 50, 60]])
A

Access elements#

A[0, 1]

Row / column#

A[0, :2]  # first row

A[:, 1]  # second column

Submatrix#

A[:, 1:]  # all rows, columns 1 to end

Views vs copies#

Slicing usually returns a view (shares data). Mutating the slice can mutate the original.

B = A[:, 1:]
B[0, 0] = 999
A  # notice A changed

If you want a copy, use .copy():

A = np.array([[10, 20, 30],
              [40, 50, 60]])
C = A[:, 1:].copy()
C[0, 0] = -1
print('C =\n', C)
print('A =\n', A)

5. Vectorized Operations#

x = np.array([1, 2, 3])
x

x * 2

x ** 2

np.sqrt(x)

np.exp(x)

Compare with a Python loop#

result = []
for value in x:
    result.append(value ** 2)
result

6. Broadcasting#

Broadcasting lets NumPy combine arrays of different shapes (when compatible).

A = np.array([[1, 2, 3],
              [4, 5, 6]])
b = np.array([10, 20, 30])
A + b  # b is broadcast across rows

Rule of thumb (simplified): compare shapes from right to left; dimensions must match or one must be 1.

7. Aggregation Functions#

Many operations can be applied to the whole array or along an axis.

A = np.array([[1, 2, 3],
              [4, 5, 6]])
A

np.sum(A)

np.mean(A)

np.sum(A, axis=0)  # sum down columns

np.sum(A, axis=1)  # sum across rows

Question: What does axis mean in your own words?

8. Matrix Operations#

Be careful: * is elementwise, while @ is matrix multiplication.

A = np.array([[1, 2],
              [3, 4]])
B = np.array([[5, 6],
              [7, 8]])
A, B

Elementwise multiplication#

A * B

Matrix multiplication#

A @ B

np.dot(A, B)

Determinant / inverse (if invertible)#

np.linalg.det(A)

h = np.linalg.inv(A)
h @ A

9. Performance Comparison#

NumPy is fast because it uses optimized compiled code under the hood.

Run these a couple times to see typical behavior.

import time

size = 1_000_000
python_list = list(range(size))
numpy_array = np.arange(size)

Python loop sum#

start = time.time()
s = 0
for i in python_list:
    s += i
end = time.time()
print('Python time:', end - start)
print('Sum:', s)

NumPy sum#

start = time.time()
s2 = np.sum(numpy_array)
end = time.time()
print('NumPy time:', end - start)
print('Sum:', s2)
