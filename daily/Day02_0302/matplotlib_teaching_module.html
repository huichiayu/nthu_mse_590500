Introduction to Matplotlib#

Audience: Students familiar with basic Python and NumPy

Learning Goals:

  • Create basic plots

  • Understand figure and axes

  • Customize labels and titles

  • Plot multiple datasets

  • Create scatter, histogram, and bar plots

  • Save figures

by GPT 5.2

0. Setup#

import numpy as np
import matplotlib.pyplot as plt

1. Your First Plot#

Plot a simple sine function.

x = np.linspace(0, 2*np.pi, 100)
y = np.sin(x)

plt.figure()
plt.plot(x, y)
plt.show()

Add labels and title#

plt.figure()
plt.plot(x, y)
plt.xlabel('x')
plt.ylabel('sin(x)')
plt.title('Sine Function')
plt.show()

2. Understanding Figure and Axes#

Matplotlib uses a figure containing axes objects.

fig, ax = plt.subplots()
ax.plot(x, np.cos(x))
ax.set_xlabel('x')
ax.set_ylabel('cos(x)')
ax.set_title('Cosine Function')
plt.show()

3. Multiple Curves on One Plot#

plt.figure()
plt.plot(x, np.sin(x))
plt.plot(x, np.cos(x))
plt.legend(['sin(x)', 'cos(x)'])
plt.title('Sine and Cosine')
plt.show()

4. Scatter Plot#

np.random.seed(0)
x = np.random.rand(50)
y = np.random.rand(50)

plt.figure()
plt.scatter(x, y)
plt.title('Random Scatter')
plt.show()

5. Histogram#

data = np.random.randn(1000)

plt.figure()
plt.hist(data, bins=30)
plt.title('Histogram')
plt.show()

6. Bar Plot#

categories = ['A', 'B', 'C', 'D']
values = [5, 7, 3, 8]

plt.figure()
plt.bar(categories, values)
plt.title('Bar Chart')
plt.show()

7. Adjusting Limits and Grid#

plt.figure()
plt.plot(x, np.sin(x))
plt.xlim(0, np.pi)
plt.ylim(-1, 1)
plt.grid(True)
plt.title('Sine (Zoomed)')
plt.show()

8. Saving Figures#

plt.figure()
plt.plot(x, np.sin(x))
plt.title('Saved Figure Example')
plt.savefig('example_plot.png')
plt.show()

  2. Create a scatter plot with 100 random points.

  3. Plot sin(x) and sin(2x) on the same graph.

  4. Save one of your figures as a PNG file.
