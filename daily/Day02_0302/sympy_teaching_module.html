SymPy Mini Teaching Module (Beginner)#

Audience: Students who know basic Python + algebra

Learning goals:

  • Create symbols and symbolic expressions

  • Simplify/expand/factor

  • Solve equations

  • Differentiate/integrate

  • Work with matrices

  • Convert symbolic expressions to numerical functions (lambdify)

by GPT 5.2

0. Setup#

!pip install Sympy

import sympy as sp
sp.__version__

Pretty printing (optional)#

In Jupyter, SymPy can display nicer math output.

sp.init_printing(use_unicode=True)

1. Symbols and Expressions#

x, y = sp.symbols('x y')
expr = (x + 2*y)**2
expr

Expand / Factor#

sp.expand(expr)

sp.factor(sp.expand(expr))

Simplify#

expr2 = (x**2 - y**2) / (x - y)
expr2

sp.simplify(expr2)

2. Substitution and Evaluation#

Use .subs to substitute values or other expressions.

expr3 = sp.sin(x) + x**2
# expr3
expr3.subs(x, sp.pi/2)

expr3.subs(x, 3).evalf()  # numeric approximation

3. Solving Equations#

Solve for unknowns with sp.solve.

One equation#

sp.solve(sp.Eq(x**2 - 5*x + 6, 0), x)

Two equations (system)#

sol = sp.solve([sp.Eq(x + y, 5), sp.Eq(x - y, 1)], [x, y])
sol

4. Calculus: Derivatives and Integrals#

Derivative#

f = sp.sin(x) * sp.exp(x)
sp.diff(f, x)

Second derivative#

sp.diff(f, x, 2)

Indefinite integral#

sp.integrate(sp.cos(x), x)

Definite integral#

sp.integrate(sp.sin(x), (x, 0, sp.pi))

Limits#

sp.limit(sp.sin(x)/x, x, 0)

5. Series Expansion (Optional but Useful)#

sp.series(sp.exp(x), x, 0, 6)  # up to x^5

6. Matrices#

A = sp.Matrix([[1, 2],
               [3, 4]])
A

A.det()

A.inv()

A.eigenvals()  # eigenvalues with multiplicities

7. Convert Symbolic → Numeric with lambdify#

This is useful when you want symbolic math but also fast numerical evaluation.

g = sp.sin(x) + x**2
# g
g_numeric = sp.lambdify(x, g, 'numpy')
# g_numeric
import numpy as np
xs = np.linspace(-2, 2, 5)
g_numeric(xs)
